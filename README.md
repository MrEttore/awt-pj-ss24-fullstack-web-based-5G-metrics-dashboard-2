## Capturing Websocket Data

### Get Started:
1. Create Environment
```bash
python3 -m venv env
````
2. Activate Environment
```bash
source env/bin/activate
```
3. Install Dependencies
```bash
pip install -r requirements.txt
```
4. Run Script
```bash
python3 capture.py
```
